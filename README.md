# Business Website Demo

This is a demo website for a fictional local business.

The goal of this project is to show that I can create a modern, responsive and practical website for small businesses such as local services, IT support, repair shops, cafés or tradesmen.

## Planned features

* Modern landing page
* Service overview
* Contact form
* Responsive design
* Basic SEO structure
* Clear call-to-action
* Business contact section

## Current status

Work in progress.
